# About This App

This super lightweight app allows you to organize photos of loyalty or access cards into custom folders using a simple and intuitive interface.

- Create as many folders as you like
- Add photos from your device gallery to any folder
- View photos as thumbnails in a clean grid layout
- Tap any image to open it in full-screen mode
- Swipe between photos and zoom in for details
- Rename or delete folders and images at any time
